##  安装文档
* 克隆仓库
* 执行 npm install
* 复制 .env.example 文件为 .env
* 执行 npm run db:init
* 执行 npm run dev

## 环境变量配置
在 .env 文件中可以配置以下环境变量：

* `DATABASE_URL`: 数据库连接地址
* `ADMIN_PASSWORD`: 管理员密码
* `NEXT_PUBLIC_USER_EXPIRATION_ENABLED`: 用户有效期功能开关
  - 设置为 `true` 时，用户管理页面会显示账号状态和有效期列
  - 设置为 `false` 或未设置时，不显示这些列
